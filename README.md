# Python Foundations for Machine Learning

## Project Overview
This repository demonstrates core Python concepts required before moving into Machine Learning engineering.  
It covers Python data structures, memory behaviour, algorithmic complexity, and Object-Oriented Programming (OOP).

The goal is to build strong programming fundamentals that support working with ML libraries such as NumPy, pandas, scikit-learn, and PyTorch.

## Concepts Covered

### Python Data Types
- list
- dictionary (dict)
- set
- tuple
- string (str)
- integers (int)
- floating-point numbers (float)

### Python Behaviour
- Mutability vs immutability
- Hashing and dictionary/set internals
- Function arguments and pass-by-object-reference
- Default argument trap
- `is` vs `==`
- Shallow copy vs deep copy

### Algorithmic Thinking
- Time complexity (O(1), O(n), O(log n), O(n log n), O(n²))
- Amortized complexity (dynamic arrays, list.append)
- Space complexity basics
- In-place vs out-of-place algorithms

### Object-Oriented Programming (OOP)
- Classes and objects
- Attributes vs methods
- `__init__` initialization
- Instance vs class variables
- Encapsulation
- Inheritance
- Method overriding
- Polymorphism
- Class methods and static methods
- Dunder methods (`__repr__`, `__len__`, `__call__`)

## Key Skills Demonstrated
- Python data structures and memory behaviour
- Algorithmic complexity analysis
- Writing efficient Python code
- Object-Oriented Programming design
- Code organisation used in ML systems

## Why This Matters for Machine Learning
Understanding Python fundamentals, algorithmic complexity, and OOP is essential for building scalable ML pipelines and working effectively with modern machine learning frameworks.
